Assignment Four: Latent Semantic Analysis (LSA) Search Engine Webpage

In this assignment, you will develop an interactive web application that implements a basic search engine using Latent Semantic Analysis (LSA). The search engine will take a user’s query, perform LSA on a pool of documents, and return the top documents based on cosine similarity.

The webpage provides the following features:

  • Document Retrieval: Users can enter a text query, and the system returns the top 5 documents based on their similarity to the query.
  • Visualization: A bar chart displays the cosine similarity of the top documents to the query.
  • User Interface: Includes an input field for the query, a “Search” button to perform document retrieval, and a results section showing the top 5 documents with similarity scores.
